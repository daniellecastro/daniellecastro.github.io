Weather Site Plan

Site Name

The site name will be "Today Feels Like..." and the domain of the site will be todayfeelslike.com

Site Purpose

The purpose of this site is to give weather information to users. The aim is to create a user friendly site that is easy to navigate and pleasing to the eye. Giving accurate weather information for the date and location stated it the simplest form possible is one the goals as well.

Site Logo

The site logo is a white cloud against a blue abckground wuth a question mark embedded into the design to tie into people's question of "what does the weather outside feel like?" It can be found at the top of this page.
